WalletWise

Aplicación web de gestión financiera desarrollada con Django, que permite a los usuarios registrar, consultar y eliminar movimientos clasificados en entradas, salidas, ahorro e inversiones. Incluye autenticación de usuarios, manejo dinámico de datos mediante AJAX, y un dashboard interactivo con cálculos en tiempo real de saldos y totales.

WalletWise

Financial management web application built with Django, allowing users to register, view, and delete transactions classified into income, expenses, savings, and investments. Features user authentication, dynamic data handling through AJAX, and an interactive dashboard with real-time balance and totals calculation.
