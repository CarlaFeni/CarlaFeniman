Travel Trace

Una app para acompañarte en tus aventuras. Guarda tus viajes, visualiza tus rutas y revive cada destino con mapas interactivos y notas personalizadas. Desarrollada en Kotlin con Jetpack Compose, integra Firebase (Auth, Firestore, Storage) para la gestión de usuarios y datos, junto con las APIs de GeoNames y Geoapify para obtener información geográfica y lugares turísticos en tiempo real.

Travel Trace

An app to accompany you on your adventures. Save your trips, visualize your routes, and relive each destination with interactive maps and personalized notes. Built with Kotlin and Jetpack Compose, it integrates Firebase (Auth, Firestore, Storage) for user and data management, along with GeoNames and Geoapify APIs to fetch geographic information and tourist spots in real time.

Bienvenida

Introduce la app TravelTrace con un botón deslizante para comenzar la experiencia.

Welcome

Introduces the TravelTrace app with a sliding button to start the experience.

Login

Permite al usuario iniciar sesión e incluye selección rápida de idioma y acceso a registro o recuperación de contraseña.

Login

Allows the user to log in with quick language selection and access to registration or password recovery.

Registro

Permite crear una nueva cuenta introduciendo nombre, correo y contraseña.

Sign Up

Allows creating a new account by entering name, email, and password.

Perfil

Muestra el resumen de los viajes del usuario con países visitados, lista de deseos y galería de fotos.

Profile

Displays a summary of the user's trips with visited countries, wishlist, and photo gallery.

Países visitados

Lista todos los países marcados como visitados con acceso a notas personales.

Visited Countries

Lists all countries marked as visited with access to personal notes.

Editor de nota para país visitado

Permite al usuario valorar y dejar un comentario personalizado sobre el país.

Note Editor for Visited Country

Allows the user to rate and leave a personalized comment about the country.

Wishlist (lista de deseos)

Muestra los países que el usuario quiere visitar con opción de marcarlos como visitados o eliminarlos.

Wishlist

Shows the countries the user wants to visit with the option to mark as visited or delete.

Pantalla de galería de fotos

En esta pantalla, el usuario puede explorar y filtrar imágenes por país para ver la galería de fotos correspondiente, seleccionando el país deseado desde un menú desplegable.

Photo Gallery Screen

On this screen, users can browse and filter images by country, selecting the desired country from a dropdown menu to view the corresponding photo gallery.

Mapa

Permite editar, eliminar y ver detalles de cada viaje guardado con una interfaz limpia y responsiva.

Map

Allows editing, deleting, and viewing details of each saved trip with a clean and responsive interface.

Diálogo opciones

Este diálogo permite al usuario gestionar un país seleccionado en el mapa, ofreciendo opciones para marcarlo como visitado, añadirlo a la wishlist, subir recuerdos o consultar información.

Options Dialog

This dialog allows the user to manage a selected country on the map, offering options to mark it as visited, add to wishlist, upload memories, or view information.

Información del país

Country Information

Displays general information about the selected country and its tourist sites categorized.

Diálogo añadir nota

Solicita al usuario si desea añadir una nota al país seleccionado.

Add Note Dialog

Prompts the user to add a note to the selected country.

Pantalla de subida de imágenes

Esta pantalla permite al usuario seleccionar y subir imágenes de un país específico, mostrando una barra de progreso para indicar el avance de la carga.

Image Upload Screen

This screen allows the user to select and upload images from a specific country, showing a progress bar to indicate the upload status

Configuración de perfil

Profile Settings
